# Robot

Provides robot interfaces and concrete integrations for supported hardware.

## Public API

- `Robot` protocol and shared interfaces
- concrete robot implementations exposed from `physicalai.robot`

## Main Modules

- `interface.py`: core robot protocols and shared types
- `connect.py`: connection helpers
- `verify.py`: validation helpers
- `so101/`: SO-101 integration
- `trossen/`: WidowX integrations
